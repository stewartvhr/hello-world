
This is a project from the Udacity Course Deep Learning that creates and trains a DCGAN network to generate new face images that look as realistic as possible. 

dlnd_face_generation.html is an html file containing the jupyter notebook that passed project specifications. 

problem_unittests.py is a python file containing unit tests that are used to test the DCGAN. 

processed_face_data.png is a png file that contains the face image data used to train the DCGAN.
