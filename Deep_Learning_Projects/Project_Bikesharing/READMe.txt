This is Project 1 completed in conjunction with the Udacity Deep Learning course. 

This project implements a neural network from scratch to predict the patterns of a bikesharing service given csv data. 

first_neural_network-271468.zip is the zip file containing the passing submission. It contains a jupyter notebook with the Python code. 

Files with the format first_neural_network-result-xxxxxx.json are previous iterations of the network. 

my_answers.py contain written answers to the written questions of the project.
